# Portfolio - Denise

Petit site statique présentant Denise (développeuse front-end).

## Aperçu
Ouvrez la page d'accueil : [index.html](index.html).

## Structure du projet
- [index.html](index.html) — page principale (contient `header`, sections `home`, `about`, `contact-form`)
- [style.css](style.css) — styles (classes importantes : `.home`, `.about`, `.contact-form`, `.logo`)
- Images :
  - [denise.jpg](denise.jpg)
  - [denise2.png](denise2.png)
  - [denise3.jpg](denise3.jpg)

## Installation / Exécution
1. Cloner ou copier le dossier sur votre machine.
2. Ouvrir [index.html](index.html) dans un navigateur.
3. (Optionnel) Utiliser l'extension Live Server de VS Code pour rechargement automatique.

## Personnalisation rapide
- Modifier le contenu HTML dans [`index.html`](index.html).
- Adapter les couleurs/typographie dans [`style.css`](style.css).
- Remplacer les images dans la racine du projet.
